## Quick Start

```bash
# 1. Clone repo and create virtual environment
git clone https://github.com/<your-org>/foolingAI-547.git
cd foolingAI-547
python -m venv venv && source venv/bin/activate

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run scripts IN ORDER from project root
python scripts/01_moabb_setup.py       # Downloads dataset (~500 MB, first run only)
python scripts/02_load_pretrained.py   # Trains and saves CSP+LDA model
python scripts/03_perturbation.py      # Tests perturbation framework
python scripts/04_integration_test.py  # End-to-end pipeline check
```

> **Important:** Always run scripts from the **project root** (`foolingAI-547/`), not from inside `scripts/`.

## Troubleshooting

| Error | Fix |
|-------|-----|
| `ModuleNotFoundError: moabb` | `pip install moabb` |
| Network error during download | Check internet; rerun script 01 (MOABB resumes partial downloads) |
| `FileNotFoundError: X_raw.npy` | Run script 01 first |
| `FileNotFoundError: model.pkl` | Run script 02 first |
| CSP convergence warning | Normal for small subject subsets; accuracy still valid |
| Low accuracy (<55%) | Expand `SUBJECTS` list in `01_moabb_setup.py` for more training data |
